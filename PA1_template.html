## [1] 2304

  1. Devise a strategy to fill in missing data. We have a dataset that contains the average activity for each time interval across the data set. My strategy is to fill the NA's with the average number of steps for that time period.

  2. Create a new dataset with the NA's filled in.

data.filled <- data.raw

# loop through the newly created dataset and replace NA's with the 5min average of that interval
for(i in 1:nrow(data.filled)) {
  if(is.na(data.filled[i,]$steps)){
    intv <- data.filled[i,]$interval
    data.filled[i,]$steps <- data.5min[data.5min$interval==intv,]$steps
  }
}
